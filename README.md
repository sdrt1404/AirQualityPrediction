# Прогнозирование концентрации PM2.5 с использованием CatBoost

## Описание проекта
Данный проект посвящён прогнозированию концентрации мелкодисперсных частиц PM2.5 в атмосферном воздухе
на основе метеорологических, временных и химических параметров окружающей среды.

В рамках проекта был реализован полный цикл работы с данными:
- исследовательский анализ данных (EDA),
- обучение модели машинного обучения,
- интерпретация результатов,
- развёртывание модели в виде REST API.

Проект выполнен в рамках курса **Data Science Junior** и соответствует требованиям классического машинного обучения.

---

## Датасет
Используется датасет **Beijing Air Quality Dataset (станция Changping)**.

Данные содержат:
- временные признаки (год, месяц, день, час),
- концентрации загрязняющих веществ (PM10, SO2, NO2, CO, O3),
- метеорологические параметры (температура, давление, точка росы, осадки, скорость и направление ветра).

Количество записей: более 35 000.

---

## Целевая переменная
**PM2.5** — концентрация мелкодисперсных частиц диаметром до 2.5 микрометра  
(единицы измерения: µg/m³).

PM2.5 является ключевым индикатором качества воздуха и представляет наибольшую опасность для здоровья человека.

---

## Используемые методы и модели

### Модель машинного обучения
- **CatBoostRegressor**

Выбор CatBoost обусловлен тем, что:
- модель эффективно работает с табличными данными,
- поддерживает категориальные признаки без one-hot кодирования,
- показывает устойчивые результаты на реальных данных.

---

## Метрики качества
Для оценки качества модели использовались:
- RMSE (Root Mean Squared Error)
- MAE (Mean Absolute Error)
- R² (коэффициент детерминации)

На тестовой выборке модель показала RMSE около **20.7**, что является хорошим результатом для задачи прогнозирования PM2.5.

---

## Интерпретация модели
Для объяснения работы модели применялись **SHAP values**:
- выполнен анализ глобальной важности признаков,
- показаны локальные объяснения для отдельных предсказаний.

Анализ показал, что наибольшее влияние на PM2.5 оказывают:
- PM10,
- NO2,
- CO,
- температура воздуха,
- скорость ветра.
